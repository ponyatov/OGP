#### Install (Linux)

```
~$ git clone -o gh https://github.com/ponyatov/OGP.git
~$ cd OGP
~/OGP$ make install
```

#### Run

* @binder: https://mybinder.org/v2/gh/ponyatov/OGP/master?filepath=OGP.ipynb

```
~/OGP$ make jupyter
```
